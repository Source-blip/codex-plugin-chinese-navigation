# Codex 插件与应用中文导航

一个可以离线打开的单文件中文导航页，用来查询 Codex 插件、可安装插件和 ChatGPT 应用/连接器。

## 在线使用

仓库启用 GitHub Pages 后，可直接在浏览器中打开导航页。

## 功能

- 收录本机在 2026 年 6 月 8 日检测到的 1547 个项目
- 包含 14 个已安装插件、28 个可安装插件和 1505 个应用/连接器
- 支持中文、英文、缩写和常用说法搜索
- 支持类型、分类、状态和图标状态筛选
- 点击卡片可查看中文介绍、英文原始介绍和官方页面
- 单文件离线运行，不依赖服务器或外部脚本
- 适配电脑和手机屏幕

## 使用方法

下载 `index.html` 后双击打开，或者访问仓库的 GitHub Pages 地址。

## 数据和准确性说明

- 目录是 2026 年 6 月 8 日的本机快照，之后可能发生变化。
- 中文介绍经过自动翻译、保守改写和程序检查，详情中保留英文原文方便核对。
- 图标按应用 ID 对应；无法取得公开官方图标的项目会明确显示“官方图标未公开”，不会拿其他应用的图标代替。
- “未安装”或“未连接”不代表应用不可用，只表示生成页面时的本机状态。

## 检测结果

`catalog-quality-report.json` 包含完整性、中文介绍、图标映射和搜索测试结果。
